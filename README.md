# 🛒 NextBuy — Responsive eCommerce Website

A sleek and fully responsive **eCommerce website** built with **pure HTML, CSS, and JavaScript**, designed to deliver a smooth and modern shopping experience.  
This project demonstrates clean UI design, interactive components, and dynamic product rendering — all without any frameworks or libraries.

---

## 🚀 Features

- 🧩 Responsive and mobile-friendly layout  
- 🛍️ Dynamic product listing (data loaded from `data.json`)  
- 🛒 Add-to-cart and remove-from-cart functionality  
- 💳 Cart total and price calculations  
- 🔍 Product search and filtering options  
- 💡 Modern UI with interactive animations  
- ⚡ 100% Vanilla JS — no frameworks used  

---

## 🧠 Tech Stack

| Technology | Purpose |
|-------------|----------|
| **HTML5** | Structure and markup |
| **CSS3** | Styling and responsiveness |
| **JavaScript (ES6)** | Logic, DOM manipulation, and interactivity |
| **JSON** | Product data source |
